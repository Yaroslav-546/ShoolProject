## 📋 О проекте

**TochkaRostaNB** — это современная веб-платформа, созданная для поддержки образовательных программ и развития личности. Проект разработан с использованием Django и развернут на PythonAnywhere.

### ✨ Основные возможности

- 📚 Управление образовательными программами
- 👥 Система пользователей и ролей
- 📊 Отслеживание прогресса обучения
- 📱 Адаптивный дизайн
- 🔒 Безопасная аутентификация
